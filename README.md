# BlazeDemo Automation Testing Project

## Project Description
This project contains automated UI tests for the BlazeDemo flight booking application. The tests cover flight search and booking functionality.

## Technologies Used
- Python
- Selenium WebDriver
- PyTest
- pytest-html
